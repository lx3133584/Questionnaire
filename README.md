# questionnaire

> 基于Vue2.0的问卷调差后台管理系统

## 技术栈

* Vue2.0（Vuex、Vue-router、Vue-resource）
* echarts
* nodejs（发布时改为php）
* ES6

## 使用

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
